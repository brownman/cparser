# cparser

* [Homepage](http://github.com/postmodern/cparser)

## Description

A pure Ruby ANSI C Parser.

## Features

* Parses ANSI C.
* Pure Ruby.

## Examples

    require 'cparser'

## Requirements

* [parslet](http://kschiess.github.com/parslet/) ~> 1.0

## Install

    $ gem install cparser
